Colossus Club Website 🚀

Official website for the Colossus Club of the Information Science & Engineering department.

This website showcases the club, its activities, upcoming events, and provides a modern online presence for the tech community.

🛠 Tech Stack

The website is built using modern web technologies:

- TypeScript – For type-safe and scalable JavaScript development
- Tailwind CSS – For fast and responsive UI styling
- Modern White Theme – Clean and minimal UI design

- Countdown launch page
- Club information and updates
- Optimized performance
